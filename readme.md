# Sistema de Producción de Triplay Ucayali

## Descripción del proyecto
Este repositorio contiene la propuesta académica y técnica de un sistema de gestión para una empresa triplayera de la región Ucayali. El sistema fue planteado para controlar el flujo principal de producción, desde el registro del lote de madera hasta el proceso de prensado, clasificación e inventario final.

## Objetivo general
Diseñar una propuesta de sistema de información que permita registrar, organizar y consultar los procesos de producción de triplay, mejorando la trazabilidad y el control de inventario.

## Estructura del repositorio
- `sql/` → scripts de base de datos, inserciones y consultas
- `figma/` → prototipos visuales y referencias HTML para mobile, tablet y desktop
- `docs/` → documentación del proyecto, explicación funcional y soporte para la exposición
